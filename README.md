Question 1: What is JSX, and why is it used?
Answer: JSX is a short form of JavaScript XML. It allow us to write HTML in javascript file. JSX basically help us to write easier code for UI, improve the code readability.
Question 2: What is the difference between State and Props?
Answer: In react props are used for passing data from parent component to child component where state is used for storing data that can change inside a component.
Question 3: What is the useState hook, and how does it work?
Answer: In react useState hook is used for storing data and managing data which can change inside a component. The method of calling useState: const [count, setCount] = useState(0); Here the initial value is 0, count stores the current value of the state and setCount is a function which update the value.
Question 4: How can you share state between components in React?
Answer: In react state can be shared in many ways like: lifting state up, using context api and state management libraries like redux.
Question 5: How is event handling done in React?
Answer: React use javascript xml for coding for that reason event handling in react is so much easy for the developers. The basic syntax for button click event handle: <button onClick = {handleClick}>Click Me</button>
Here onClick is a event name which is in camel case and handleCLick is a function that runs when the event occurs.
